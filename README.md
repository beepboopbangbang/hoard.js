# Hoard CLI

A small (zero dependency) Node.js CLI utility clone & hoard repositories quickly

## Node.js Install

```sh
npm install -g hoard-cli

yarn global add hoard-cli

pnpm add -g hoard-cli
```
[Install from NPM](https://www.npmjs.com/package/hoard-cli)

## CLI Usage
```sh
# show help
hoard -h

# hoard all repositories under `beepboopbangbang` user or org
hoard https://github.com/beepboopbangbang/hoard.js
```
